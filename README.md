# plataforma-para-an-lisis-de-movimiento
La plataforma para análisis de movimiento en tiempo real se creo utilizando 
Python y utiliza cámaras Intel RealSense D435I Y D455 para la detección 
y seguimiento en tiempo real.
La plataforma para el análisis de señales se creo en MATLAB y necesita de 
de dos archivos de tipo CSV o xls para comparar los resultados deseados
