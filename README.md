# plataforma-para-an-lisis-de-movimiento
La plataforma para analisis de movimiento en tiempo real se creo utilizando 
con Python y utiliza cámaras Intel RealSense D435I Y D455 para la deteccion 
y seguimiento en tiempo real.
La plataforma para el analisis de señales se creo en MATLAB y necesita de 
de dos archivos de tipo CSV o xls para comparar los resultados deseados
